## ğŸ¯ Tips & Tricks

1. **Natural Language:** JARVIS samajhta hai natural Hindi/Hinglish. Formal hone ki zaroorat nahi.

2. **Contact Names:** Contacts ko Hindi names se bhi call kar sakte ho:
   - "Mummy", "Papa", "Bhai", "Behen", "Dada", "Dadi"

3. **Time Format:** Time 24-hour format mein bolo:
   - "7 baje" = 07:00
   - "Shaam 5 baje" = 17:00
   - "Raat 9 baje" = 21:00

4. **Multiple Commands:** Ek saath multiple commands nahi de sakte. Ek command complete hone do, phir dusra do.

5. **Permissions:** Pehli baar use karne par permissions maangega. Allow karna zaruri hai.
